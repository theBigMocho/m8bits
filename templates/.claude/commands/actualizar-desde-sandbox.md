# Actualizar desde Sandbox

Ejecuta el script batch que actualiza los componentes desde el sandbox.

**Comando a ejecutar:**
```
C:\parse\proyectos\apps\promet\arp-components\actualizar-desde-sandbox.bat
```

Al finalizar, muestra:
- El output completo del comando
- Cualquier error que haya ocurrido
- Confirmación de que la actualización se completó exitosamente
