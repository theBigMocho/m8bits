# Incrementar Versión

Lee la versión actual del archivo `Promet/arp-components/arp-turnos-component/version.txt`, incrementa el número de patch (tercer dígito), y escribe la nueva versión al archivo.

Después de incrementar la versión, muestra:
- Versión anterior
- Nueva versión
- Ruta del archivo actualizado

No hagas commit ni push. Solo incrementa la versión.
